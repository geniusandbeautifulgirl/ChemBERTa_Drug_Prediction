# ChemBERTa_Drug_Prediction
Custom drug solubility prediction model using ChemBERTa.

## Files
- train_model.py: Train a prediction model on your drug dataset
- predict_drug.py: Predict solubility for new molecules
- drug_solubility_model.pkl: Saved trained model

## Usage
1. Run training:
   python train_model.py

2. Predict new molecules:
   python predict_drug.py

## Model
ChemBERTa-zinc250k-v1
